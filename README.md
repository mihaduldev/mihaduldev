## Engineering principles I follow

- **Keep systems understandable** — simple architecture beats clever complexity.
- **Design for change** — business requirements evolve, so code should be easy to extend.
- **Security is not optional** — authentication, authorization, validation, and safe defaults matter.
- **Automate repeatable work** — CI/CD, Docker, scripts, and AI tools should reduce manual effort.
- **Document the why** — good engineering is not only code, it is shared understanding.
